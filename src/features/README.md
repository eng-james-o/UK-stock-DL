# Features Module

Implementation of various features used for stock prediction, including technical indicators and sentiment analysis.

## Submodules

- **technical_indicators.py**: Contains functions for calculating technical indicators such as RSI, ROC, ATR, CCI, ADX, AO, OBV, and Williams %R.
- **sentiment.py**: Provides functionality for sentiment analysis of financial news headlines using the VADER sentiment analyzer.

## Key Indicators

- **RSI**: Relative Strength Index
- **MACD**: Moving Average Convergence Divergence (implied via MA/EMA)
- **ATR**: Average True Range
- **Sentiment**: Daily compound sentiment score from news headlines.
