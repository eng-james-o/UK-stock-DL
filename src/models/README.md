# Models Module
Implementation of various model architectures for stock price prediction.

- `base_model.py`: Abstract base class for all models.
- `model_cl.py`: CNN-LSTM architecture.
- `model_gru.py`: GRU architecture with optional Attention.
- `model_var.py`: Vector Autoregression (VAR) model.
- `model_gan.py`: Generative Adversarial Network (GAN) for price prediction.
- `train.py`: Utilities for data splitting and training.
